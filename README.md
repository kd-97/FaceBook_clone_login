# facebook_clone

Flutter project for Creating a Clone Facebook Login page using Flutter.


## Components_Used
🖊️Stateless Widget
🖊️Box Decoration
🖊️Box Shadow
🖊️Icons
🖊️Elevated Buttons for
   -Create New Account
   -Forgot Password
   -Login with Phone Number
   -Login with Google
🖊️TextFormFeild
